Программа предазначена для сортировки Excel таблицы:
	Ищутся совпадения первых слов в колонке "Артикул". При их наличии складываются колонки "КОЛ-ВО".
	Т.е. в итоге имеется файл с суммарным количеством объектов с "одинаковым" Артикулом.
	Дополнительно заполняются столбцы "Товар", "Размер", "ЦЕНА БЕЗ ОРГ" содержимым из строки Артикула найденного первым.
	
Баги:
1. Улетает в ошибку если в ключевых колонках встречаются пустые ячейки
Как минимум последняя строка только с общей суммой выбила прогу.
2. Не сообщает о происходящих ошибках.
3. Не сообщает о записи файла.
